# 智慧水产养殖平台 · Smart Aquaculture Platform

基于 Web 的智慧水产养殖管理系统，提供水质实时监测、设备远程控制、视频监控、历史数据分析、报警通知等一站式解决方案。

## 功能模块

| 模块    | 说明                           |
| ----- | ---------------------------- |
| 总览仪表盘 | 水质参数实时展示、变化趋势曲线、塘口地图分布、设备清单  |
| 水质监测  | 溶解氧、水温、pH值、盐度、氨氮、亚硝酸盐等参数实时监测 |
| 设备控制  | 增氧机、水泵、投料机、照明、温控等设备远程启停      |
| 视频监控  | 塘口实时画面监控                     |
| 历史数据  | 水质历史记录查询与导出                  |
| 设备绑定  | 网关、传感器、监测箱的绑定配置              |
| 报警设置  | 多通道报警通知（短信/邮箱/电话）            |
| 账号管理  | 多角色权限管理、操作记录审计               |

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript, Chart.js, Leaflet.js
- **设计**: 毛玻璃风格 (Glassmorphism)，适配桌面端与移动端

```

├── 海南世电_默认3-1.svg         # 公司品牌 Logo
```

## 使用

直接在浏览器打开 `index.html` 或 `mobile.html` 即可预览。
