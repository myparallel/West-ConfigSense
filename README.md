## 中文

WEST公司基于指标数据池和AI驱动的农林牧渔"灵配智控"WEB应用平台软件和手机APP软件HTML效果模拟。

### 项目简介

这是一个智慧水产养殖管理平台的前端模拟项目，展示了WEST公司在农林牧渔领域的智能化解决方案。

### 功能模块

- **实时监控**：水质参数实时监测与展示
- **设备控制**：远程控制增氧机、水泵等设备
- **数据分析**：历史数据查询与趋势分析
- **报警通知**：多通道报警通知（短信/邮箱/电话）
- **视频监控**：塘口实时画面监控

### 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **设计**: 毛玻璃风格 (Glassmorphism)
- **适配**: 桌面端与移动端

### 项目结构

```
├── index.html                    # 桌面端主页面
├── ConfigSense_preview.html      # 预览页面
├── ConfigSense_mobile.html       # 移动端页面
├── ConfigSense_需求规格说明书_v1.0.md  # 需求规格文档
├── ConfigSense_需求规格说明书_v1.5.pdf # 需求规格PDF
├── ConfigSense灵配智控_APP产品Logo.png # 产品Logo
├── WEST公司LOGO.png              # 公司Logo
├── 水产养殖业发展.pdf            # 行业分析报告
└── 参考历史：智慧养殖APP/        # 参考资料
```

### 使用

直接在浏览器打开 `index.html` 或 `ConfigSense_preview.html` 即可预览。

## English

WEST's AI-driven "ConfigSense" WEB application platform and mobile APP HTML mockup for agriculture, forestry, animal husbandry, and fishery based on metric data pools.

### Project Overview

This is a frontend simulation project for a smart aquaculture management platform, demonstrating WEST's intelligent solutions in the agriculture, forestry, animal husbandry, and fishery fields.

### Feature Modules

- **Real-time Monitoring**: Real-time water quality parameter monitoring and display
- **Device Control**: Remote control of aerators, water pumps, and other devices
- **Data Analysis**: Historical data query and trend analysis
- **Alarm Notifications**: Multi-channel alarm notifications (SMS/Email/Phone)
- **Video Surveillance**: Real-time pond video monitoring

### Tech Stack

- **Frontend**: HTML5 + CSS3 + JavaScript
- **Design**: Glassmorphism style
- **Adaptation**: Desktop and mobile

### Project Structure

```
├── index.html                    # Desktop main page
├── ConfigSense_preview.html      # Preview page
```
